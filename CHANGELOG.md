# Changelog

## 0.1.0 - 2026-05-26

- Initial public release.
- Added Prettier 3 plugin entry point for `.njk` and `.nunjucks` files.
- Added placeholder-based formatting through Prettier's HTML parser.
- Added post-processing for common Nunjucks block indentation and set tags.
- Added tests for formatting, idempotency, raw blocks, placeholder collisions,
  and Nunjucks delimiters inside string literals.
